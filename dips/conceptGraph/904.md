DIP-904
======

set, superset
------------------------------

Following the convention of [DIP-900](900.md), new `wordTypes` will be introduced: `set` and `superset`.

Example:

```json
{
  "wordData": {
    "wordTypes": ["set"],
  },
  "setData": {
  },
  ...
}
```

```json
{
  "wordData": {
    "wordTypes": ["superset"],
  },
  "supersetData": {
  },
  ...
}
```
