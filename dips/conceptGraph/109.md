DIP-109
======

superset
------------------------------

Following the convention of [DIP-102](102.md), a new `wordType` will be introduced: `superset`.

## Declaration

```json
{
  "wordData": {
    "wordTypes": ["wordType"],
  },
  "wordTypeData": {
    "slug": "superset"
  },
  ...
}
```

## Example:

```json
{
  "wordData": {
    "wordTypes": ["superset"],
  },
  "supersetData": {
  },
  ...
}
```
