DIP-904
======

set
------------------------------

Following the convention of [DIP-900](900.md), a new `wordType` will be introduced: `set`.

## Declaration

```json
{
  "wordData": {
    "wordTypes": ["wordType"],
  },
  "wordTypeData": {
    "slug": "set"
  },
  ...
}
```

Example:

```json
{
  "wordData": {
    "wordTypes": ["set"],
  },
  "setData": {
  },
  ...
}
```
